# 设计初衷

这个项目是我第一个接后端的项目，其目的只是为了学习后端的编写。“日历”只是实现的载体，前后端交互的实现才是这个项目的目的。这个日历允许添加/修改每一天的心情状态（表示为-100~100间的数值）和备注来表示一天的状态。

### 设计思路

为了白嫖免费的后端，所有的后端程序都加载在[Cloudflare](https://cloudflare.com)上，静态网页依旧是挂载在[Github](https://github.com)上。由于不需要存储复杂的内容，后端存储我选择了Cloudflare的D1数据库（SQLite数据库）。由于没有域名，目前后端加载在默认位置。*到时候有时间可以去搞一个 ovo*
