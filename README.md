## 基本介绍
这是一个以个人网页为主的仓库。基本所有的网页都是由`html`、`css`、`js`编写，<!-- 且基本都是由AI生成的， -->而其他部分以`python`为主，大部分为个人编写的。<i>为避免不必要的麻烦，所有课程相关的代码均不会出现于此仓库中。</i>

## 网页架构
网页架构如下：
```
├── icon/
├── project/
├── shade/
└── index.html
```

### 详细架构：
`icon`中存储所有主要图标，包括主图标、项目图标等；

`project`中包含所有项目，包括在线项目与离线项目。大多数项目为兴趣使然。项目页面标准可以在[项目README](./project/README.md)中找到；

`shade`中包含我画的画 (//ω//)；

`index.html`即为网页主页。
